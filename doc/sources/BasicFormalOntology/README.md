## Configuration properties

| Property       | Values     | Description                                              |
|----------------|------------|----------------------------------------------------------|
| forceExport    | true/false | Force graph export even if nothing changed               |
| ignoreObsolete | true/false | Ignore OBO ontology entries if they are flagged obsolete |
